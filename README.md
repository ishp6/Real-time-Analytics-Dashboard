# Real-Time Analytics Dashboard
 
A responsive business analytics dashboard with live data simulation and interactive charts. Built with vanilla JavaScript — no frameworks.
 
---
 
## What it does
 
- Live metric updates every 2 seconds
- Interactive line and doughnut charts via Chart.js
- Real-time activity feed with animations
- Start, stop, and reset monitoring controls
- Keyboard shortcuts: `Ctrl+S` to toggle, `Ctrl+R` to reset
 
---
 
## Stack
 
HTML5 · CSS3 · Vanilla JavaScript (ES6+) · Chart.js
 
---
 
## Run it
 
Just open `index.html` in a browser. No installs, no build step.
 
```bash
git clone https://github.com/ishp6/Real-time-Analytics-Dashboard.git
cd Real-time-Analytics-Dashboard
# open index.html
```
